String常用方法总结

一、String类

      String类在java.lang包中，java使用String类创建一个字符串变量，字符串变量属于对象。java把String类声明为final类。String类对象创建之后不能修改，由0或多个字符组成，包含在一对双引号之间。


二、String类对象的创建

字符串声明：String stringName;
字符串创建：stringName = new String(字符串常量);或stringName = 字符串常量;
